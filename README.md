# disco-push

[Pushover][pushover] notification when someone enters a [Discord][discord] channel

# Usage

## Setup Discord Bot

### Required Bot Permissions

- [x] Manage Channels

## Set envronment variables

- `DISCORD_BOT_TOKEN`
- `CHANNEL_ID`
- `PUSHOVER_USER`
- `PUSHOVER_TOKEN`

## Execute

```console
python disco-push.py
```

[pushover]: https://pushover.net
[discord]: https://discord.gg
